```

   GPIO2  (3) (4)  5V

     GND  (9) (10) GPIO15 ← IN2 (Left Motor Backward)

  GPIO27  (13) (14) GND
  GPIO22  (15) (16) GPIO23
    3V3  (17) (18) GPIO24

```

## Connection Summary

### Motor Control Pins:
- ENA (GPIO 14, Pin 8) → Left Motor Speed Control
- IN1 (GPIO 18, Pin 12) → Left Motor Forward
- IN2 (GPIO 15, Pin 10) → Left Motor Backward
- ENB (GPIO 17, Pin 11) → Right Motor Speed Control
- IN3 (GPIO 22, Pin 15) → Right Motor Forward
- IN4 (GPIO 27, Pin 13) → Right Motor Backward

### Power Connections:
- L298N VCC → Pi 5V (Pin 2)
- L298N GND → Pi GND (Pin 6)
- L298N VIN → LM2596S OUT+ (6V)

### Control:
- START BUTTON → GPIO 16 (Pin 36)
